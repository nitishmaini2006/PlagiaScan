# 🛡️ PlagiaScan

AI Powered Text Similarity & Plagiarism Detection Web App.

## 🚀 Features

✅ Compare TXT files  
✅ Compare PDF files  
✅ Compare DOCX files  
✅ Similarity percentage analysis  
✅ Interactive modern UI  
✅ Real-time plagiarism detection  
✅ Progress visualization  

---

## 🛠️ Tech Stack

- Python
- Streamlit
- difflib
- PyPDF2
- python-docx
